wavves POS

wavves, a simple POS system built on Express.js and Handlebars.js, is a lightweight business management application designed to help businesses establish an easy to use POS. There are three main functions: a) creating new transactions, b) modifying the products that are available for sale, and c) viewing previous transactions. chefMEup

chefMEup is a powerful application that allows a user to quickly find recipes that match the ingredients they have in their fridge.

chefMEupchefMEup is a simple to use application that allows the user to mark the ingredients they have in their fridge and quickly view all recipes that match. Each match shows the user all their matching ingredients and also those they are missing. With the Whisk API, users can order these missing ingredients straight to their door via grocery delivery services like Instacart. To save ingredients or create recipes, users can create a chefMEup account by signing in through Google via PassportJS. With over 1000 ingredients from multiple countries, our aspiring chefs can create and share recipes to the growing chefMEup community! All ingredients, recipes, and user accounts are stored in a MySQL database (via the Sequelize ORM) hosted on Heroku. Application routes are handled via Express.js and pages are rendered via Handlebars.js.

Design Framework: Bootstrap
