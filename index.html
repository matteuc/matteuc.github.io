My Work

Pokemon Stay

A Pokemon-Go inspired game designed for physically challenged trainers to explore the world of Pokemon.

Pokemon Stay

Through the PokéAPI, Pokemon Stay is populated by 649 Pokemon sprites. Via Firebase's Realtime Database, trainers are able to save their current location and collection of Pokemon. The Google Maps API allows trainers to travel across the globe - discovering, fighting, and catchin' them all.

Design Framework: Bootstrap

GifTastic

A single page application that allows the user to search and save GIFs from GIPHY.

GifTastic

Through the GIPHY API, GifTastic pulls a fixed number of GIFs based on the entered search term. Users can favorite GIFs, which the app stores in local storage to allow for data persistence. Popular searches can be transformed into buttons to quickly access your favorite GIFs.

Design Framework: Material Design
